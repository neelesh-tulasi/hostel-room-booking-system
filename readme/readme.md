# 🏠 Hostel Room Booking System

## 📖 Overview

The Hostel Room Booking System is a Flask-based web application developed to automate hostel room allocation for students. It allows students to register, log in, book hostel rooms, and download booking receipts. Administrators can manage hostels, rooms, students, and bookings through a dedicated dashboard.

---

## ✨ Features

### Student Module
- Student Registration
- Student Login
- Edit Profile
- Hostel Booking
- Room Selection
- Booking History
- Download Booking Receipt
- Logout

### Admin Module
- Admin Login
- Dashboard Statistics
- Add Hostel
- Edit Hostel
- Delete Hostel
- Add Room
- Edit Room
- Delete Room
- View Students
- Edit Student
- Delete Student
- View Bookings
- Cancel Booking

## 🚀 Installation

1. Clone the repository

```
git clone https://github.com/neelesh-tulasi/hostel-room-booking-system.git
```

2. Install dependencies

```
pip install -r requirements.txt
```

3. Run the application

```
py app.py
```

4. Open in your browser

```
http://127.0.0.1:5000
```
